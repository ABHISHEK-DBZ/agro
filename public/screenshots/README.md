# Screenshots for PWA Manifest

This directory contains screenshots for the PWA manifest. These screenshots are displayed in app stores and when users try to install the PWA.

## Required Screenshots:
1. **dashboard.png** - 1280x720 (wide) - Dashboard with weather and crop information
2. **disease-detection.png** - 750x1334 (narrow) - AI-powered disease detection
3. **weather.png** - 750x1334 (narrow) - Live weather updates
4. **market-prices.png** - 750x1334 (narrow) - Real-time market prices

## How to Create Screenshots:
1. Open the app in your browser
2. Navigate to each page
3. Take screenshots using browser dev tools (Ctrl+Shift+I)
4. Use device emulator for mobile views (750x1334)
5. Use desktop view for wide screenshots (1280x720)
6. Save screenshots with exact names mentioned above
