# 🔧 Smart Krishi Sahayak APK - Troubleshooting Guide

## 📱 APK Not Opening? यहाँ समाधान है!

### 🚨 **Common Issues & Solutions**

#### 1. **APK Download करने के बाद Open नहीं हो रहा**

**समस्या**: APK file corrupt हो सकती है या WebView dependency issue है

**समाधान**:
```
✅ Step 1: पुराना APK delete करें
✅ Step 2: नया APK download करें (Build ID में update check करें)
✅ Step 3: Installation से पहले "Unknown Sources" enable करें
✅ Step 4: अगर फिर भी issue है तो Simple Mode का option मिलेगा app में
```

#### 2. **Installation Failed Error**

**समस्या**: Android security settings या storage issue

**समाधान**:
```
📱 Settings > Security > Unknown Sources > Enable
📱 Settings > Apps > Package Installer > Permissions > Enable
📱 Clear कम से कम 100MB storage space
📱 Restart phone और फिर try करें
```

#### 3. **App Crashes Immediately**

**समस्या**: Device compatibility या memory issue

**समाधान**:
```
🔄 Phone restart करें
🧹 Background apps close करें
📶 Internet connection check करें
🔧 Latest APK download करें (improved version)
```

#### 4. **WebView Loading Failed**

**समस्या**: Network या WebView component issue

**समाधान**:
```
✅ App में "Simple Mode" button दबाएं
✅ यह direct browser में open कर देगा
✅ Full functionality available होगी
✅ सभी features काम करेंगे
```

## 🔧 **Step-by-Step Installation**

### 📱 **For Android Devices:**

1. **Download APK**
   ```
   🔗 EAS Dashboard से latest build download करें
   📱 या QR code scan करें (अगर available है)
   ```

2. **Enable Unknown Sources**
   ```
   ⚙️ Settings > Security > Unknown Sources > Enable
   📱 या Settings > Apps > Package Installer > Allow
   ```

3. **Install APK**
   ```
   📁 Downloads folder में APK file find करें
   🔸 File पर tap करें
   ✅ "Install" confirm करें

   ```
