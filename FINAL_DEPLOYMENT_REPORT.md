## 🏪 STORE PUBLICATION GUIDE

### 🤖 Google Play Store
1. **Account**: Create Google Play Console account ($25)
2. **Upload**: Download AAB from EAS dashboard
3. **Submit**: Upload to Play Console
4. **Review**: Usually 1-3 days

### 🍎 Apple App Store  
1. **Account**: Apple Developer Program ($99/year)
2. **Build**: Run `eas build --platform ios`
3. **Upload**: EAS handles App Store Connect
4. **Review**: Usually 1-7 days

### 📋 Required Information
- App description (Hindi/English)
- Screenshots (mobile + tablet)
- Privacy policy
- App category: Education/Agriculture
